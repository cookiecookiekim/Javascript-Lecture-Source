01. 자식 노드 탐색

노드 탐색

        요소 노드를 취득한 후 취득한 요소를 기점으로
        부모 or 형제 or 자식을 탐색해야 하는 경우가 존재할 수 있음.

공백 텍스트 노드

        HTML 요소 사이의 스페이스, 탭, 줄바꿈 등의 공백 문자는
        텍스트 노드를 생성한다.

자식 노드 탐색하기

  1. childNodes : 자식 노드(요소, 텍스트)를 탐색하여 NodeList 객체 반환
  2. firstChild : 첫 번째 자식 노드(요소, 텍스트) 반환
  3. lastChild : 마지막 번째 자식 노드(요소, 텍스트) 반환
  1. children : 자식 노드 중 요소(태그)만 탐색하여 HTMLCollection 객체 반환
  2. firstElementChild : 첫 번째 자식의 요소 노드 반환
  3. listElementChild : 마지막 번 째 자식의 요소 노드 반환
