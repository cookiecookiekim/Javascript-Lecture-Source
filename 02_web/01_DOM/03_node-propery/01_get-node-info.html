01. 요소 노드의 텍스트 조작

nodeValue

        nodeValue는 getter, setter가 모두 존재하는 프로퍼티이다.
        nodeValue 프로퍼티 참조 시 노드 객체의 값 (텍스트 노드의 텍스트) 반환.
        여기서 주의할 점은 텍스트가 아닌, 요소 노드인 경우 null 반환.
변경 전

        textContent도 getter, setter가 모두 존재하는 프로퍼티이다.
        하지만 nodeValue와 다른 점은 요소 노드의 텍스트와
        모든 자손 노드의 텍스트를 모드 취득하거나 변경 가능하다.
