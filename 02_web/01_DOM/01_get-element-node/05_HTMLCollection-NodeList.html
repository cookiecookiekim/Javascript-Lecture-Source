05.HTMLCollection과 NodeList

        HTMLCollection과 NodeList는 여러 개의 결과 값을
        담기 위한 DOM 컬렉션 객체의 종류.
        이 두가지는 모두 유사 배열 객체이면서 이더러블이다.

        노드 객체의 상태 변화를 실시간으로 반영하는
        살아있는 DOM 컬렉션 객체이다.

querySelectorAll() 메서드가 반환하는 컬렉션 객체
