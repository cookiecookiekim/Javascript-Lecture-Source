05. 노드 복사, 교체, 삽입

        clonNode([deep : true | false]) 
        노드의 사본을 생성하여 반환.
        매개변수에 true를 전달하면 노드를 깊은 복사하여 
        모든 자손 노드가 포함된 사본을 반환해주고,
        false를 전달하면 얕은 복사를 하여 노드 자신만의 사본 생성.
