01. open, alert, confirm, prompt 메서드

    자바스크립트 언어는 브라우저 상에서 동작하는 언어이다.
    자바스크립트가 web 상에서 동작할 때 사용할 수 있는 기능이 있다.

    web 상에서 가장 최상위 객체는 window 객체이다.
    window
        - DOM (document 등)
        - BOM (location , navigator 등)
        - JavaScript (Array , Function , Object 등)
    최상단의 window 객체는 브라우저 창을 대변하고
    브라우저를 제어할 수 있는 메서드를 제공한다.

    window는 전역 객체이기 때문에 생략이 가능하다.
    ex → window.alert() → alert()

open

    window.open(url, name, params) 메서드로 새로운 창 열 수 있다.

alert

    window.alert(message) 메서드는 확인 버튼을 가지며,
    메세지를 지정할 수 있는 경고 상자를 띄워준다.

confirm

        window.confirm(message) 메서드는 확인과 취소,
        두 버튼을 가지고 있으며, 메시지를 지정할 수 있게 된다.

        ☆★ 반환값이 존재하며 확인을 누르면 true,
        취소 or ESC를 누르면 false를 반환한다.

prompt(명령 날리다)

        window.prompt(message, default)
        - message : 사용자가 어떤 값을 입력해야 하는지 안내문구
        - default : 텍스트 입력 필드에 기본 값(문자열)
        → 반환값은 사용자 입력 값이며,
          취소를 누르거나 ESC키를 누르게 되면 null 값 리턴.
