03. addEventListener 방식으로 이벤트 등록

        EventTarget.addEventListener 메서드를 사용하여 이벤트 등록 가능.
    EventTarget.addEventListener('eventType', functionName[, useCaoture]);
                                                    
    첫 번째 인자 : 이벤트의 타입을 지정하며, on 접두사는 생략 ex) ('click')
    두 번째 인자 : 이벤트 핸들러 함수 등록
    세 번째 인자 : 생략이 가능하며, 이벤트 전파와 관련 (추후에 다룰 예정.)
